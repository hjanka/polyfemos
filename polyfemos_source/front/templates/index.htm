Polyfemos web
=============

Polyfemos provides a state of health data processing web application for observatory measurement equipment monitoring. The program features data visualization, statictical analysis, data coverage and realtimeness monitoring and problem detecting and locating.

The application provides a realtime status information of stations' conditions and informs about the possible problems with the network.

    The application features:
  • State of health map, a visual presentation of the general status of the network.
  • State of health table, similar to the map but delivers more information.
  • Data coverage browser to scan the data coverage.
  • Statistical summaries of different state of health parameters of the stations.
  • Plot browser, an interactive plotter to visualize the state of health data.
