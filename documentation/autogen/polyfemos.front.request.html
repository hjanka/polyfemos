polyfemos.front.request.filepath(filepath, extension, force=False)[source]¶

Returns the filepath with the given extension if it exists. If not, check if the ‘retro’ version is available and returns that. If both are unavailable, returns an empty string.

Parameters

  • filepath (str) – original filepath without the extension

  • extension (str) – file extension, e.g. .stf, .csv or .alert

  • force (bool) – If True, returns filepath with the given extension and if it does not exist, returns an empty string.

Return type

str
